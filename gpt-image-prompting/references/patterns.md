# Pattern Library

## 1) Data visualization / infographic
Use when the image must explain a claim visually.

Core ingredients:
- domain or topic
- headline and subhead
- chart type
- color coding for each series
- annotation callout
- source line
- explicit canvas shape

Short pattern:
```text
Create a clean editorial infographic about [topic]. Headline: "[headline]". Subhead: "[subhead]". Show a [chart type] with [series A] in [color A] and [series B] in [color B]. Include one short annotation highlighting [insight]. Add source line: "[source]". Minimal background, modern magazine style, readable typography, [aspect ratio].
```

## 2) Hyper-detailed realistic scene
Use when density, realism, and many readable objects matter.

Core ingredients:
- camera body / lens / exposure style
- time of day
- crowd/object density
- material behavior
- specific readable text in scene
- foreground / midground / background details

Short pattern:
```text
Ultra-detailed real photograph of [scene], shot on [camera/lens], [time/light]. Dense lived-in environment with [key subjects]. Materials behave realistically: [surface details]. Include clearly readable signs/text: [text list]. Cinematic realism, sharp focus on [priority], natural imperfections, [aspect ratio].
```

## 3) Flash portrait realism
Use when the goal is believable human photography.

Core ingredients:
- camera style
- pose
- face / hair / clothing
- skin texture details
- flash behavior
- realism constraints

Short pattern:
```text
Realistic flash photograph, [framing], [subject description], [pose]. Natural skin texture, visible pores, fine hair detail, realistic fabric behavior, direct flash with hard-edged shadow, accurate anatomy, photorealistic color, no plastic skin, [aspect ratio].
```

## 4) Pinterest / fashion look
Use for outfit, lifestyle, social-media visuals.

Core ingredients:
- outfit logic
- pose logic
- environment simplicity
- platform vibe
- texture realism

Short pattern:
```text
Pinterest-style fashion image of [subject] wearing [outfit]. Pose: [pose]. Clean lifestyle background, social-media editorial feel, realistic clothing folds and accessories, polished but natural, [aspect ratio].
```

## 5) Split-panel comparison
Use for before/after, zoom/detail, process, makeup, feature comparisons.

Core ingredients:
- exactly how many panels
- what each panel shows
- consistent lighting/style
- what changes between panels

Short pattern:
```text
Create a [two-panel/three-panel] image. Panel 1: [view one]. Panel 2: [view two]. Keep lighting and subject consistent. Emphasize [difference]. Realistic detail, clean composition, [aspect ratio].
```

## 6) Annotated image
Use when the image itself should teach.

Core ingredients:
- base image or scene
- labels to overlay
- typography style
- arrow/callout behavior
- analytical tone

Short pattern:
```text
Create an annotated visual of [subject]. Overlay labels for [label list]. Use clean typography, precise callouts, restrained colors, educational editorial style. Keep the underlying image readable and the annotations accurately placed.
```

## 7) Clean tech product visual
Use for app, AI, startup, SaaS, device, or abstract product storytelling.

Core ingredients:
- hero object
- spatial composition
- palette
- light behavior
- product surface quality
- UI or data hints

Short pattern:
```text
Clean tech hero visual for [product/topic]. Central focus on [hero object]. Soft neutral background, subtle gradients, crisp industrial design, minimal but premium composition, restrained palette with [accent color], gentle reflections, modern startup aesthetic, [aspect ratio].
```

## 8) Editorial comic explainer
Use for narrative posts and article covers.

Core ingredients:
- one clear scene
- simplified but expressive forms
- limited palette
- room for headline overlay
- conceptual metaphor

Short pattern:
```text
Editorial comic illustration about [topic]. Show [scene/metaphor]. Magazine-style composition, expressive characters/forms, limited palette, clear focal point, clean negative space for title, smart and modern rather than childish, [aspect ratio].
```
