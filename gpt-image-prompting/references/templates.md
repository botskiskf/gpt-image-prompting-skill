# Ready Templates

## Universal prompt template
```text
Create a [format] showing [main subject]. Composition: [composition]. Lighting: [lighting]. Key details: [details]. Text visible in image: [text or none]. Style: [style]. Color palette: [palette]. Background: [background]. Mood: [mood]. Aspect ratio: [ratio].
```

## Telegram post cover — clean tech
```text
Create a clean tech editorial cover image about [topic]. Central focus on [hero object/metaphor]. Minimal composition, soft light background, crisp modern forms, subtle depth, restrained palette with [accent color]. Premium startup aesthetic, readable negative space for headline overlay, no clutter, [aspect ratio].
```

## Telegram post cover — editorial comic
```text
Create an editorial comic style cover illustration about [topic]. Show [metaphor/scene]. Clear focal subject, modern magazine composition, expressive but tidy shapes, limited palette, slight wit, clean negative space for headline overlay, [aspect ratio].
```

## Product mockup
```text
Create a premium product mockup of [product]. Three-quarter view, soft studio lighting, realistic materials, subtle reflections, clean neutral background, modern tech branding feel, highly legible screen/interface, minimal props, [aspect ratio].
```

## Realistic portrait
```text
Create a realistic portrait photo of [subject]. [framing] shot, [pose/expression], [camera/look]. Natural skin texture, believable hair and fabric detail, clean lighting, realistic anatomy, no over-stylization, [aspect ratio].
```

## Comparison / before-after
```text
Create a two-panel comparison image. Left: [state A]. Right: [state B]. Same subject, consistent angle and lighting. Clearly emphasize the difference in [feature]. Clean composition, realistic detail, [aspect ratio].
```

## Infographic card
```text
Create a square infographic card about [topic]. Headline: "[headline]". Subhead: "[subhead]". Visualize [data point/relationship] using [chart type]. One short annotation for [insight]. Minimal editorial layout, readable typography, restrained colors, source line at the bottom.
```

## Annotated explainer
```text
Create an annotated explainer image of [subject]. Show [base image]. Add precise labels for [label list]. Use thin callout lines, clean typography, modern educational editorial style, high clarity, uncluttered layout, [aspect ratio].
```

## Short prompt expansion checklist
Add only what is needed:
- camera / lens
- angle / framing
- lighting behavior
- surface or skin texture
- required on-image text
- palette
- negative space
- realism constraint
