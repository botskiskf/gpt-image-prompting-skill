# Готовые шаблоны — RU

## Универсальный шаблон
```text
Создай [формат], где показан [главный объект/сцена]. Композиция: [композиция]. Свет: [свет]. Ключевые детали: [детали]. Текст в изображении: [текст или «без текста»]. Стиль: [стиль]. Палитра: [палитра]. Фон: [фон]. Настроение: [настроение]. Соотношение сторон: [ratio].
```

## Обложка Telegram-поста — clean tech
```text
Создай clean tech editorial cover image на тему [тема]. Центральный фокус на [главный объект/метафора]. Минималистичная композиция, мягкий светлый фон, четкие современные формы, легкая глубина, сдержанная палитра с акцентом [цвет]. Премиальный startup aesthetic, читаемое свободное место под заголовок, без визуального шума, [соотношение сторон].
```

## Обложка Telegram-поста — editorial comic
```text
Создай editorial comic style cover illustration на тему [тема]. Покажи [метафора/сцена]. Ясный главный объект, современная журнальная композиция, выразительные, но аккуратные формы, ограниченная палитра, легкая ирония, чистое свободное место под заголовок, [соотношение сторон].
```

## Product mockup
```text
Создай премиальный product mockup для [продукт]. Вид в три четверти, мягкое студийное освещение, реалистичные материалы, тонкие отражения, чистый нейтральный фон, современное tech-brand ощущение, хорошо читаемый экран/интерфейс, минимум реквизита, [соотношение сторон].
```

## Реалистичный портрет
```text
Создай реалистичную portrait photo: [человек/персонаж]. Кадрирование: [кадр]. Поза/выражение: [поза/эмоция]. Стиль камеры: [камера/look]. Естественная фактура кожи, правдоподобные волосы и ткань, чистый свет, реалистичная анатомия, без чрезмерной стилизации, [соотношение сторон].
```

## Сравнение / before-after
```text
Создай two-panel comparison image. Слева: [состояние A]. Справа: [состояние B]. Один и тот же объект, одинаковый угол и свет. Четко подчеркни разницу в [особенность]. Чистая композиция, реалистичная детализация, [соотношение сторон].
```

## Инфографическая карточка
```text
Создай квадратную infographic card на тему [тема]. Заголовок: «[заголовок]». Подзаголовок: «[подзаголовок]». Визуализируй [данные/связь] через [тип графика]. Добавь одну короткую аннотацию: [инсайт]. Минималистичная editorial-верстка, читабельная типографика, сдержанные цвета, источник внизу.
```

## Аннотированный explainer
```text
Создай annotated explainer image для [объект/тема]. Покажи [базовое изображение]. Добавь точные подписи для [список меток]. Тонкие callout-линии, чистая типографика, современный образовательный editorial-стиль, высокая ясность, без перегруза, [соотношение сторон].
```

## Шаблон для Telegram/соцсетей — clean tech
```text
Создай clean tech обложку для Telegram-статьи о [тема]. Визуальная метафора: [метафора]. Композиция минимальная и премиальная, фон светлый нейтральный, один акцентный цвет [цвет], современное ощущение AI/SaaS/startup, аккуратная глубина, место под заголовок, без лишних деталей, 16:9.
```

## Шаблон для Telegram/соцсетей — editorial comic
```text
Создай editorial comic обложку для Telegram-статьи о [тема]. Сцена: [сцена/метафора]. Стиль современного бизнес-медиа, не мультяшно-детский. Чистый фокус, ограниченная палитра, легкая ирония, место под заголовок, аккуратная детализация, 16:9.
```

## Чеклист расширения короткого промпта
Добавляй только нужное:
- камера / объектив
- угол / кадрирование
- поведение света
- материал / фактура / кожа
- обязательный текст на изображении
- палитра
- свободное место под заголовок
- ограничение по реализму
