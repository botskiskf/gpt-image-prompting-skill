# Библиотека паттернов — RU

## 1) Инфографика / визуализация данных
Когда нужно объяснить тезис через картинку, график или карточку.

Что указать:
- тема / домен
- заголовок и подзаголовок
- тип графика
- цвета для серий данных
- один главный вывод
- источник
- формат холста

Шаблон:
```text
Создай чистую editorial-инфографику на тему [тема]. Заголовок: «[заголовок]». Подзаголовок: «[подзаголовок]». Покажи [тип графика] с [серия A] цветом [цвет A] и [серия B] цветом [цвет B]. Добавь короткую аннотацию с главным выводом: [инсайт]. Внизу укажи источник: «[источник]». Минималистичный фон, современная журнальная верстка, читабельная типографика, [соотношение сторон].
```

## 2) Реалистичная детальная сцена
Когда нужна плотная, живая, фотореалистичная картинка с множеством деталей.

Что указать:
- камера / объектив / стиль съемки
- время суток и свет
- плотность объектов / людей
- материалы и фактуры
- читаемый текст в кадре
- передний / средний / дальний планы

Шаблон:
```text
Ультрадетальная реалистичная фотография сцены [сцена], снятая на [камера/объектив], [время/свет]. Живая среда с множеством деталей: [ключевые объекты/люди]. Реалистичные материалы: [фактуры/поверхности]. В кадре должен быть четко читаемый текст: [список текста]. Кинематографичный реализм, резкий фокус на [главный объект], естественные несовершенства, [соотношение сторон].
```

## 3) Реалистичный портрет со вспышкой
Когда нужна правдоподобная human-фотография.

Что указать:
- тип камеры / съемки
- поза
- лицо / волосы / одежда
- фактура кожи
- поведение вспышки
- ограничения реализма

Шаблон:
```text
Реалистичная фотография со вспышкой, [кадрирование], [описание человека], [поза/выражение]. Естественная фактура кожи, видимые поры, реалистичные волосы и ткань, прямой жесткий свет вспышки, четкая тень на фоне, правильная анатомия, фотореалистичный цвет, без пластиковой кожи, [соотношение сторон].
```

## 4) Pinterest / fashion / lifestyle
Для одежды, lifestyle-картинок, соцсетей.

Что указать:
- образ / outfit
- поза
- простая среда
- vibe платформы
- реалистичные складки и аксессуары

Шаблон:
```text
Pinterest-style fashion image: [человек] в образе [одежда]. Поза: [поза]. Чистый lifestyle-фон, ощущение современной соцсети/editorial-съемки, реалистичные складки ткани и аксессуары, отполировано, но естественно, [соотношение сторон].
```

## 5) Split-panel / сравнение
Для before/after, zoom/detail, этапов, сравнения функций.

Что указать:
- точное количество панелей
- что в каждой панели
- одинаковый свет и стиль
- что меняется

Шаблон:
```text
Создай изображение из [двух/трех] панелей. Панель 1: [вид 1]. Панель 2: [вид 2]. Сохрани одинаковый объект, угол и освещение. Четко подчеркни разницу в [параметр]. Чистая композиция, реалистичные детали, [соотношение сторон].
```

## 6) Аннотированное изображение
Когда картинка должна объяснять сама себя.

Что указать:
- базовый объект / сцена
- список меток
- стиль типографики
- стрелки / callouts
- обучающий тон

Шаблон:
```text
Создай аннотированное изображение [объект/сцена]. Добавь подписи для [список меток]. Используй чистую типографику, точные callout-линии, сдержанные цвета, образовательный editorial-стиль. Базовое изображение должно оставаться читаемым, подписи размещены точно.
```

## 7) Clean tech product visual
Для AI, SaaS, приложений, startup, абстрактных tech-обложек.

Что указать:
- главный объект / метафора
- композиция
- палитра
- свет
- качество поверхностей
- hints интерфейса / данных

Шаблон:
```text
Clean tech hero visual для [продукт/тема]. Центральный фокус на [главный объект/метафора]. Мягкий нейтральный фон, тонкие градиенты, четкий современный дизайн, минимальная премиальная композиция, сдержанная палитра с акцентом [цвет], легкие отражения, startup aesthetic, [соотношение сторон].
```

## 8) Editorial comic explainer
Для обложек статей, сторителлинга и объясняющих постов.

Что указать:
- одна понятная сцена
- простые выразительные формы
- ограниченная палитра
- место под заголовок
- визуальная метафора

Шаблон:
```text
Editorial comic illustration на тему [тема]. Покажи [сцена/метафора]. Журнальная композиция, выразительные аккуратные формы, ограниченная палитра, ясный фокус, чистое свободное место под заголовок, умно и современно, не детский стиль, [соотношение сторон].
```
