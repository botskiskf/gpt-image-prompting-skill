# GPT Image Prompting Skill

An OpenClaw/Agent skill for building strong image-generation prompts from reusable GPT-Image-2 style patterns.

Includes:
- prompt formula and workflow
- English and Russian pattern libraries
- ready templates for clean tech, editorial comic, product mockups, portraits, split-panel comparisons, infographics, and annotated explainers
- source notes distilled from a public prompt collection

## Install

Copy `gpt-image-prompting/` into your OpenClaw skills directory.
